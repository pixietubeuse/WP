Breviter is an elegant, bold WordPress theme for creative bloggers and professionals. It's based on clean, beautiful design and robust code. Use it to give your blog or site a fresh and modern look and feel.

== How to install ==

You can install the theme through WordPress dashboard by going to "Themes" > "Install themes" and searching for "Breviter".

Or you can download the theme archive, unzip it and move the unzipped contents to the "wp-content/themes" folder of your WordPress installation. You will then be able to activate the theme.

You will be able to customize your theme via WordPress theme customizer.
